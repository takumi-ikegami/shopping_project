# shopping_project

成果物 shopping のソースコードです。

LAMP 環境で MVC モデルになっております。<br>
PHPのバージョンは8.0です。

表示には Twig を使用しております。<br>
デザイン部分にはSCSSを使用しております。

環境構築には Docker を使用しております。

以下の機能を実装済みです。

一般会員向け機能<br>
・会員情報登録機能<br>
・ログイン機能<br>
・商品表示機能（一覧・詳細）<br>
・カート機能（登録・削除・購入）<br>
・お問い合わせ<br>

管理者機能<br>
・会員情報一覧表示（登録・変更・削除）<br>
・商品情報一覧表示（登録・変更・削除）<br>
・お問い合わせ情報確認<br>
・商品購入情報確認<br>
・各情報の登録日付によるソート<br>
